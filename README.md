## My Portfolio

A personal portfolio website that encapsulates my personal projects and other works. 

## Tech Stack 
- [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`]
- [TailwindCSS](https://github.com/tailwindlabs/tailwindcss)

## Getting Started

1. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

2. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

## Deployment

I used [Vercel Platform](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme) from the creators of Next.js.
