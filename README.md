# My Portfolio

A personal portfolio website showcasing my projects and other works.

## Getting Started

1. Run the development server:
```bash
   npm run dev
```

2. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result. The page auto-updates as you edit `app/page.js`.

## Deployment

Deployed on [Vercel](https://vercel.com), the platform from the creators of Next.js.
